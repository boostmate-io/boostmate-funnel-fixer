# Project Memory

## Core
- Multi-tenant auth: Main Accounts -> Sub Accounts (Workspaces). Data scoped to `sub_account_id`.
- Stack: Supabase (Edge Functions), React Flow, Recharts, Gemini Flash, Firecrawl. No full page reloads on workspace switch.
- Design: Primary #6246ffff. Headings: Manrope, Body: Inter. Modals: z-20 close btn with round bg-muted/80.
- Localization: Default English. Dutch available via dashboard settings only.
- 1 account = 1 project. Standard accounts have 1 workspace.
